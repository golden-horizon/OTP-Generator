# OTP-Generator
6-digit OTP (One-Time Password) and display it to the user. The OTP will expire after 5 seconds, and a new OTP will be generated when the user clicks the "Generate New OTP" button.

A simple **React-based One-Time Password (OTP) Generator**. This project generates a 6-digit OTP, starts a 5-second countdown, and disables the button while the OTP is valid. Once expired, the user can generate a new OTP.

## Features
- Generates a **random 6-digit OTP**.
- Countdown timer that disables the button while OTP is active.
- Expired OTP message.
- Clean and modern UI with responsive design.


## Technologies Used
- React
- JavaScript (ES6+)
- CSS3
